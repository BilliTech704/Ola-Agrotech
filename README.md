# Ola-Agrotech

A modern portfolio website for Olanrewaju Trading Ltd, built with Next.js and Tailwind CSS.

## Features

- Responsive design for mobile and desktop
- Company portfolio showcasing trading services
- Contact form for inquiries
- Professional business layout

## Services

- Agricultural Products Export
- Brand Promotion Services
- Industrial Goods Importation

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Vercel Deployment
